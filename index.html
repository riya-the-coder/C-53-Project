JSX and React Native Code

A declarative style(- we tell what are the different components we want in the application) of coding would make the app more organized. For that we have a new programming language called JSX and a new mobile development framework called React Native to help build powerful and complex apps in an easy way. JSX is a combination of both HTML and Javascript. JSX and React Native code in an online editor called 'snack' 'Snack' is an online editor written by Expo, which allows us to write and compile React Native code in an online environment. We can also preview the output of the code using Snack.
